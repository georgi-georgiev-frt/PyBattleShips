PyBattleShips
=============
Console application - python version of BattleShips game.

Installation
------------
Download this repository
    
    git clone https://github.com/georgi-georgiev-frt/PyBattleShips.git
    
Run the binary file

    cd PyBattleShips
    ./bin/battleships

Running tests

    ./bin/runtests
